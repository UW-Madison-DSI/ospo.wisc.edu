### Project leader:
James E. Pustejovsky (pustejovsky@wisc.edu)
### Project description:
[lmeInfo](https://github.com/jepusto/lmeInfo) is an R package that provides
analytic derivatives and information matrices for fitted linear mixed effects
(lme) models and generalized least squares (gls) models estimated using
nlme::lme() and nlme::gls(), respectively. The package includes functions for
estimating the sampling variance-covariance of variance component parameters
using the inverse Fisher information. The package does not currently provide
methods for calculating degrees of freedom that are standard in other software
packages (such as SAS) but this is a top priority on the roadmap for the
project. An intern would be responsible for:
1. Developing functions that calculate degrees of freedom for the supported
models, following known formulas.
2. Writing unit tests and examples to verify correct implementation.

### Intern needs:
R programming, matrix algebra, knowledge of statistical theory at advanced undergraduate level.
