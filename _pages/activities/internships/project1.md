### Project Description:
The vast majority of coarse-grained simulations focus on the dynamics of
spherical, isotropic particles. Such approaches neglect molecular geometry, which can in turn
yield incorrect simulation results. Over the past two years, the Cersonsky lab has developed a
method to use machine learning to accurately predict how anisotropic particles interact within a
system, paving the way towards accurate, machine-learning driven, shaped-particle simulation.
However, there is still software development required to enable these advances.
Along with the team in the Cersonsky lab, interns will design, document, and test a software
library that:
- From an atomistic configuration, generates a library of candidate coarse-grained “beads”
(particles that represent multiple atoms).
- Performs efficient calculation of anisotropic machine-learned descriptors for end users.
- Enables seamless training of machine-learned potentials in anisotropic systems.
- Can be incorporated into popular simulation engines (such as LAMMPS) for easy
training and simulation.
- Takes advantage of available hardware (e.g. GPU acceleration) to maximize
performance.
Interns will learn how to design Machine Learning APIs that can be both used for end-users and
integrated into established software. They will also learn good scientific software development
practices (Version control/collaborative code development, documentation, testing, modular
code).
