### Project leader:
James E. Pustejovsky (pustejovsky@wisc.edu)

### Project description:
[clubSandwich](https://github.com/jepusto/clubSandwich) is a popular R package
(in the top 10 percent of all R package downloads) that provides several
cluster-robust variance estimators for ordinary and weighted least squares
linear regression models, two-stage least squares regression models, and
generalized linear models. Several adjustments are incorporated to improve
small-sample performance. The package follows an object-oriented design to allow
for straight-forward extension to other classes of regression models. Several
models are not currently supported but are on the roadmap. An intern would be
responsible for:
1. Learning about a statistical model that is not currently
supported.
2. Developing required methods so that the clubSandwich works for the
model.
3. Writing unit tests and examples to verify correct implementation.

### Intern needs:
R programming, knowledge of statistical theory at advanced undergraduate level.
